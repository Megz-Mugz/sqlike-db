End goal TLDR:
- client sends query over tcp connection
- server gets query, parsers and typechecks
- assuming valid statement, server generates x86 code w/ .s file
- server links it and executes the machine code
- server takes the result of the executed code and sends back to client
- result is then displayed to client
- keep repeating above steps until client disconnects
